# Typing Test

*Take a typing speed test with this **Typing Test** over Python 3.9*


Improve your typing speed with this typing minigame. It has fifty short phrases in two different languages: English and Spanish. 


## Installation

This code does not work with external libraries so no installation is necessary (Python 3 only in case it is not installed).



##### Clone this repository [GitHub](https://github.com/metalpoch/typing_test#)
```bash
git clone https://github.com/metalpoch/typing_test
```
##### Enter the new directory 
```bash
cd typing_test
```

## Use
##### Run using [Python 3](https://www.python.org/downloads/release/python-395/)
```bash
python app.py
```
##### When asking the language, answer with "en" or "es" 
```bash
Select a language:
(en) - English
(es) - Spanish
R:     
```

### *That's all. Break yout personal record...*
